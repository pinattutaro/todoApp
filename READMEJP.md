# React Todoアプリ

これは、React、TypeScript、Viteで構築された機能豊富なシングルページのTodoアプリケーションです。スムーズで直感的なユーザーエクスペリエンスに重点を置き、個人のタスク管理のための包括的なツールセットを提供します。

## ✨ 機能

### コア機能
- **Todo管理**: タスクの作成、編集、削除。各タスクには説明、期限、完了ステータスが含まれます。
- **プロジェクトベースの整理**: タスクをカスタマイズ可能なプロジェクトに整理します。各プロジェクトには、視覚的に区別しやすいように、固有の名前、色、アイコンがあります。
- **データ永続化**: すべてのタスクとプロジェクトはブラウザの`localStorage`に自動的に保存され、セッション間でデータが安全に保たれます。

### UI & UX
- **日付中心のダッシュボード**: メインビューには、選択した日付のタスクがプロジェクトごとにグループ化されて表示されます。
- **インタラクティブな日付ナビゲーション**:
    - ヘッダーコンポーネントには、素早いナビゲーションのために5日間のビュー（今日±2日）が表示されます。
    - 各日付のバッジは、期限切れのタスク数を示します。
    -特定の日付にジャンプするための完全なデイトピッカーが利用可能です。
- **動的ナビゲーションパネル**:
    - 異なるビュー（`リスト`、`カレンダー`、`期限切れ`）を切り替えます。 *現在、リストビューのみが実装されています。*
    - サイドパネルのプロジェクトをクリックして表示/非表示を切り替えることで、タスクをフィルタリングします。
    - プロジェクトをドラッグアンドドロップして、好みに合わせて並べ替えます。
- **リッチなタスクブロック**:
    - タスクは期限に基づいて色分けされます：期限切れ、今日が期限、または将来が期限。
    - ワンクリックでタスクを完了としてマークします。完了したタスクは取り消し線が引かれ、もう一度クリックすることでリストからクリアできます。
    - 直感的なインラインエディタを使用すると、タスクの説明、期限、親プロジェクトをその場で変更できます。
- **タスクのドラッグアンドドロップによる並べ替え**: ドラッグアンドドロップでプロジェクトリスト内のタスクを簡単に並べ替えます。
- **柔軟なタスク作成**:
    - グローバルな`+`ボタンで、デフォルトのプロジェクトにタスクをすばやく追加します。
    - プロジェクト固有の`+`ボタンで、関連するプロジェクトに直接タスクを追加します。

## 🛠️ 技術スタック
- **フレームワーク**: React
- **言語**: TypeScript
- **ビルドツール**: Vite
- **アニメーション**: スムーズなドラッグアンドドロップインタラクションのためのFramer Motion。
- **スタイリング**: 動的なインラインスタイルを持つプレーンなCSS。

## 🚀 はじめに

### 前提条件
- Node.js
- npm または yarn

### インストールと実行
1. リポジトリをクローンします。
2. 依存関係をインストールします:
   ```bash
   npm install
   ```
3. 開発サーバーを実行します:
   ```bash
   npm run dev
   ```
4. ブラウザを開き、提供されたローカルURLに移動します。
