# React Todoアプリ

このアプリケーションは、React、TypeScript、Viteで構築された、多機能なシングルページのTodoアプリケーションです。スムーズで直感的なユーザー体験に重点を置いた、個人向けタスク管理のための包括的なツールセットを提供します。

## ✨ 機能

### コア機能
- **Todo管理**: タスクの作成、編集、削除が可能です。各タスクには、説明、締め切り日、完了ステータスが含まれます。
- **プロジェクトベースの整理**: タスクをカスタマイズ可能なプロジェクトに整理します。各プロジェクトには、視覚的に区別しやすいように、ユニークな名前、色、アイコンが設定されます。
- **データ永続化**: すべてのタスクとプロジェクトは、ブラウザの`localStorage`に自動的に保存されるため、セッション間でデータが失われることはありません。

### UI & UX
- **日付中心のダッシュボード**: メインビューには、選択した日付のタスクがプロジェクトごとにグループ化されて表示されます。
- **インタラクティブな日付ナビゲーション**:
    - ヘッダーコンポーネントには、素早くナビゲーションできるように5日間のビュー（今日 ± 2日）が表示されます。
    - 各日付のバッジは、その日に締め切りを迎えるタスクの数を示します。
    - 特定の日付にジャンプするための完全な日付ピッカーが利用可能です。
- **動的なナビゲーションパネル**:
    - 異なるビュー（`リスト`、`カレンダー`、`期限切れ`）を切り替えます。 *現在、リストビューのみが実装されています。*
    - サイドパネルでプロジェクトをクリックして表示/非表示を切り替えることで、タスクをフィルタリングします。
    - プロジェクトをドラッグ＆ドロップして、好みの順序に並べ替えることができます。
- **リッチなタスクブロック**:
    - タスクは締め切りに応じて色分けされます：期限切れ、今日が締め切り、または未来。
    - ワンクリックでタスクを完了としてマークします。完了したタスクには取り消し線が引かれ、再度クリックすることでリストから削除できます。
    - 直感的なインラインエディタにより、タスクの説明、締め切り、親プロジェクトをその場で変更できます。
- **ドラッグ＆ドロップによるタスクの並べ替え**: ドラッグ＆ドロップでプロジェクトリスト内のタスクを簡単に並べ替えることができます。
- **柔軟なタスク作成**:
    - グローバルな`+`ボタンで、デフォルトのプロジェクトに素早くタスクを追加できます。
    - プロジェクト固有の`+`ボタンで、関連するプロジェクトに直接タスクを追加できます。

### 🚧 計画中の機能
- **カレンダービュー**: 月間カレンダーでタスクを視覚的に確認する方法。
- **期限切れビュー**: 期限切れのすべてのタスクを確認するための専用セクション。

## 🛠️ 技術スタック
- **フレームワーク**: React
- **言語**: TypeScript
- **ビルドツール**: Vite
- **アニメーション**: スムーズなドラッグ＆ドロップ操作のためのFramer Motion。
- **スタイリング**: 動的なインラインスタイルを持つプレーンなCSS。

## 🚀 はじめに

### 前提条件
- Node.js
- npm または yarn

### インストールと実行
1. リポジトリをクローンします。
2. 依存関係をインストールします:
   ```bash
   npm install
   ```
3. 開発サーバーを実行します:
   ```bash
   npm run dev
   ```
4. ブラウザを開き、提供されたローカルURLにアクセスします。
